nano commnad:

write nano <filename>

to save a nano file press control+S on keybord
to exit a nano file press control+X on the keybord



vi commnad:

write vi <filename>

to save a vi file press Press ESC key-> then : -> w ->hit enter  on keybord
to exit a vi file press q after writing w  on the keybord.. :wq

to start writing content in vi editor press i on the keybord to get inside insert mode.

:wq! for focefully saving and quiting.



cd - : takes you back to last exited folder.
